# Rou.video 使用指南

> 版本 v3.4 · 从零开始下载 rou.video 视频，只需 3 分钟。

---

## 安装

```
1. 浏览器安装 Tampermonkey（Chrome/Edge 商店搜索即可）
2. 打开 Tampermonkey → 添加新脚本 → 粘贴 scripts/rou-video-downloader-v3.user.js 全部内容 → Ctrl+S 保存
3. 配置代理 127.0.0.1:10808（推荐 Chrome 安装 SwitchyOmega，
   新建规则 *.rou.video → 127.0.0.1:10808；或用 Windows 系统代理）
4. 访问 rou.video 任意视频 → 右上角出现红色面板 → 点 [⚡ 下载 MP4]
```

> **提示**：如果页面右上角没有出现面板，按 F12 → Console，搜索 `[RouDL]` 看是否有日志。
> 完全没有日志说明脚本没运行，检查 Tampermonkey 中脚本是否启用。

---

| 按钮 | 功能 |

| **💾 保存 .m3u8** | 保存播放列表，供 yt-dlp/ffmpeg 手动下载 |

## 常见问题

### Q1: 右上角没有下载面板？
→ F12 Console 搜 `[RouDL]`。无日志→脚本未运行（检查 Tampermonkey 是否启用+ `@match` 是否匹配）。有日志但显示"等待"→等几秒，脚本会自动检测。

### Q2: 只有 404p，没有高清？
网站的清晰度由服务端根据账户权限控制，脚本无法绕过。**解决方案**：注册 rou.video 账户并登录，重新访问视频，脚本会自动获取权限内最高画质。

### Q3: 片段下载失败？
脚本会跳过失败的片段继续下载。如果大量失败，参照日志中的 HTTP 状态码：
- **403** → auth token 过期，刷新视频页面重试
- **401** → 需要登录
- **timeout** → 检查代理连接

### Q4: 下载文件名是什么？
自动提取视频标题作为文件名，格式 `{标题}_{画质}p.mp4`（去掉了网站后缀和非法字符）。

### Q5: 下载的能播放吗？
可以。文件是完整的 HLS 流合并为 `.mp4`，Windows 自带播放器、PotPlayer、VLC 等均可直接打开。

### Q6: 跨域权限提示怎么跳过？
脚本已添加 `@connect *`，应不再弹出。如仍出现，点一次"允许"即可。

## 高清获取说明

网站 `__NEXT_DATA__` 中列出了 480p/720p/1080p，但 m3u8 URL 由服务端的 EV 加密字段按用户权限返回。**客户端脚本无法伪造更高权限**。

**建议操作**：
1. 在 rou.video 注册账户并登录
2. 重新访问视频，脚本自动检测权限获取对应画质
3. 如仍只有 404p，说明当前账户无更高权限（可能需要 VIP）
